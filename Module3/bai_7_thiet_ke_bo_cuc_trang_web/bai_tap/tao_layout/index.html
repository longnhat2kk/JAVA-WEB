Giới thiệu về Java

Java là gì ?

Java hiện nay là ngôn ngữ lập trình phổ biến trong các ứng dụng web, di động, desktop, game. Nó được tạo ra vào năm 1995 và các ứng dụng viết bằng Java đang chạy trên 3 tỉ ứng dụng khác nhau

Tại sao sử dụng Java ?

  • Ứng dụng Java chạy trên đa nền tảng khác nhau như Window, Mac, Linux.
  • Nó là ngôn ngữ phổ biến trên thế giới được nhiều người sử dụng.
  • Java thường rất dể học cho các bạn ban đầu đi học lập trình
  • Java thì hoàn toàn miễn phí chúng ta không phải trả tiền
  • Các ứng dụng viết bằng Java mang tính bảo mật cao
